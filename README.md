# DrawMe
# Project Overview
DrawMe is a collaborative online platform where users can engage in creative drawing sessions in real-time. The platform allows multiple users to draw on a shared canvas while communicating through an integrated live chat feature. DrawMe is designed to foster creativity, collaboration, and interaction among users, making it perfect for artists, friends, and teams looking to brainstorm visually.

# Features
- Real-time Collaborative Drawing: Multiple users can draw simultaneously on a shared canvas. All users can see each other's contributions in real-time.
- User-Friendly Interface: Simple, intuitive design ensures that users can start drawing and chatting without any technical hurdles.
- Multi-Device Support: DrawMe is responsive and works smoothly across various devices, including desktops, tablets, and smartphones.
- Color and Brush Tools: Users can choose from a variety of colors and brush sizes to customize their drawing experience.
- Undo/Redo Functionality: Mistakes are easily corrected with undo and redo options, allowing for a smooth drawing experience.
